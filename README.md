## SpringBoot_Init 
- RestAPI 기본 입문  
- 게시판 기능 구현 

### 게시판 주요 기능 
1. 글 쓰기(/board/save)
2. 글 목록(/board/)
3. 글 조회(/board/{id})
4. 글 수정(board/update{id})
5. 글 삭제(board/delete/{id})
6. 페이징 처리(/board/paging)

### Framework 
- SpringBoot(with Initalizer)
- JDK17
- MySQL
- JPA
- Thymeleaf


### 설정 사항 
- PORT = 5432
